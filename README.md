Механика «сначала ты жертва и прячешься, а потом находишь оружие и становишься охотником».

### 1. Основной цикл 
Делаем систему уровней (например, "Нейронная сеть" или "Зараженный сервер"). На каждом уровне (этаже) есть две фазы:
 * Фаза выживания (Стелс): Игрок появляется без оружия. На уровне бродят вирусы. Цель игрока - ориентируясь по лабиринту, найти «Ядро антивируса» (аналог пистолета/ключа). В этой фазе игрок может только убегать, прятаться за укрытиями или использовать окружение (например, запирать двери).
 * Фаза зачистки (Экшен): Как только игрок берет Ядро, его спрайтовая моделька меняется, и он получает возможность стрелять. Теперь уровни нужно зачистить от определенного количества вирусов, чтобы открылся портал на следующий этаж.
### 2. Процедурная генерация лабиринтов
Уровни-лабиринты нужно генерировать случайным образом.
 * Двумерная карта представляется в виде матрицы.
 * Для генерации самих коридоров можно применить алгоритмы из теории графов. Если представить сетку как граф, то алгоритм поиска в глубину (DFS) с возвратом идеально подойдет для создания запутанных, но гарантированно проходимых лабиринтов.
### 3. Развитие и прокачка (RPG-элементы)
Так как это рогалик, между уровнями (или при подборе лута) герой должен становиться сильнее:
 * Виды оружия: Вместо одного пистолета, «Ядро» каждый раз может быть разным. Дробовик (бьет на 3 клетки конусом, отталкивает), Снайперский луч (бьет через всю карту по прямой, но требует ход на перезарядку), Гранатомет (наносит урон по площади, но может задеть и тебя).
 * Модули передвижения: Возможность делать рывок на 2 клетки («Даш»), проходить сквозь одну стену раз в 10 ходов или ставить голограмму-обманку для вирусов.
 * Пермадет: Если вирус добирается до игрока (или заканчиваются очки прочности), система "перезагружается" - игрок теряет модули и начинаешь с первого уровня, но, возможно, сохраняет глобальную валюту для покупки стартовых бонусов.
### 4. Разнообразие вирусов (Враги)
Нужна экосистема:
 * *Слизни:* Медленные (ходят раз в 2 хода), но оставляют за собой ядовитый след.
 * *Охотники:* Быстрые, двигаются точно к игроку по кратчайшему пути (здесь как раз пригодится волновой алгоритм или A* для их ИИ).
 * *Танки:* Неуязвимы спереди, их нужно обходить и стрелять в спину.
