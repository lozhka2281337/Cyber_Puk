# Демонстрация процедурной генерации подземелий (BSP Tree + Pygame)

## 🚀 Особенности реализации

* **BSP Разбиение**: Карта рекурсивно делится на случайные сектора с контролем пропорций сторон (чтобы избежать слишком узких комнат).
* **Г-образные коридоры**: Соединение комнат происходит по центральным точкам через родительские узлы дерева, что обеспечивает 100% связность всей карты.
* **Масштабирование тайлов**: Все расчеты геометрии происходят в тайловой сетке, а отрисовка автоматически масштабируется через коэффициент `TILE_SIZE`.
* **Интерактивный просмотр**: Возможность мгновенно перегенерировать карту «на лету» без перезапуска скрипта.

---

## 📂 Структура файлов в директории

* `generate.py` - Логика генератора. Содержит класс `Leaf`, отвечающий за разделение зон, построение комнат и расчет геометрии коридоров.
* `main.py` - Точка входа. Инициализирует окно Pygame, обрабатывает ввод пользователя и занимается визуализацией сгенерированной карты.

---

## 🎮 Управление в демонстрации

Запустите файл `main.py`:
```bash
python main.py
```

В открывшемся окне доступны следующие горячие клавиши:
* `SPACE` (Пробел) - Сгенерировать **новую карту** с другими случайными параметрами.
* `ESCAPE` (Esc) или закрытие окна на крестик - **Выход** из приложения.

---

## ⚙️ Настройка параметров под себя

В верхней части файла `generate.py` вы можете изменить константы, чтобы полностью поменять архитектуру подземелья:

* `MIN_LEAF_SIZE` - Минимальный размер зоны. Чем он меньше, тем больше комнат поместится на карте.
* `MIN_ROOM_SIZE` / `MAX_ROOM_SIZE` - Контролируют диапазон случайного размера комнат.
* `HALL_WIDTH` - Ширина проходов между комнатами (в тайлах).
* `TILE_SIZE` - Масштаб отображения одного тайла в пикселях на экране.
